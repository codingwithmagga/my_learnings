## Practice Problem 1.1

### A

$T_{old} = 25h$  
$T_{new} = \frac{1000 km}{100 \frac{km}{h}} + \frac{1500 km}{150 \frac{km}{h}} = 20h$

$S=\frac{T_{old}}{T_{new}} = \frac{25h}{20h} = 1.25$

### B

$S=\frac{25h}{T_{new}} = 1.67 => T_{new} = 15h$

$T_{new} = \frac{1000 km}{100 \frac{km}{h}} + \frac{1500 km}{v} = 15h$  
$ => v = \frac{1500km}{15h - 10h} = 300\frac{km}{h}$
