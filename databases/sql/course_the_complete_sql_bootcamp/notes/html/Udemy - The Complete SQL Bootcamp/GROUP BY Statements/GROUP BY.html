GROUP BY


  • Describe→A SQL clause that groups rows with the same values in specified columns into a summary row.
  • Basic syntax→SELECT category_col, AGG(data_col) FROM table GROUP BY category_col
  • The GROUP BY clause must appear right after a {{1492825440530936::FROM}} or {{1492825440530936::WHERE}} statement.
  • In the SELECT statement, columns must either have {{7488734170323531::an aggregate function}} or be in the {{6775170167643532:: GROUP BY}} call.
  • WHERE statement should not refer to the {{18076557036896956::aggregation result}}, for this use {{18076557036896956::HAVING}}.
  • If you want to sort results based on the aggregate, make sure to {{035386134482203735::reference the entire function}}.
